## 🌟 Характеристики

### Страници

#### 1. **Начало** (index.html)
- Hero секция с информация за проекта
- Видове външни запаметяващи устройства (6 карти)
- Рискове и мерки за защита
- Timeline на проекта
- Footer с контакти

#### 2. **Графики** (graphics.html)
- 6 интерактивни графики с Chart.js:
  - Скорост на четене/запис (Bar chart)
  - Капацитет спрямо цена (Scatter plot)
  - Пазарен дял (Doughnut chart)
  - Издръжливост (Horizontal bar)
  - Енергопотребление (Line chart)
  - Честота на използване (Polar area)
- Сравнителна таблица
- Анализ и изводи

#### 3. **Контакти** (contact.html)
- Контактна форма с валидация:
  - Име (задължително)
  - Email (задължително, валидация)
  - Телефон (опционално, автоформатиране)
  - Тема (задължително)
  - Съобщение (задължително, 10-500 символа)
  - Checkbox за поверителност
- Real-time валидация
- Character counter
- Контактна информация
- FAQ секция (accordion)

#### 4. **Автор** (author.html)
- Профил на автора
- Технически умения с прогрес бар
- Образование
- Проекти (3 карти)
- Постижения
- Цитат
- CTA секция

### 🎨 Дизайн и стилове

- **Цветова схема:**
  - Primary: #2563eb (синьо)
  - Secondary: #1e40af (тъмно синьо)
  - Success: #10b981 (зелено)
  - Danger: #ef4444 (червено)
  - Warning: #f59e0b (оранжево)

- **Responsive дизайн:**
  - Мобилни: 320px+
  - Таблети: 768px+
  - Десктоп: 1024px+
  - Широки екрани: 1440px+

- **Accessibility:**
  - WCAG AA съвместимост
  - Контраст 4.5:1
  - Семантичен HTML
  - ARIA атрибути
  - Keyboard navigation
  - Screen reader friendly

### 💻 JavaScript функционалност

#### main.js
- Мобилна навигация (hamburger menu)
- Smooth scroll
- FAQ accordion
- Scroll animations
- Scroll to top бутон
- Lazy loading на изображения
- Active страница в навигацията

#### charts.js
- 6 типа графики с Chart.js
- Responsive графики
- Интерактивни tooltips
- Анимация при зареждане

#### contact.js
- Real-time валидация
- Регулярни изрази за email, телефон
- Character counter
- Auto-formatting на телефон
- Error messages
- Success message

## 🚀 Как да стартирате

### Метод 1: Python HTTP Server
```bash
cd /workspaces/diploma/src
python3 -m http.server 8000
```
Отворете: http://localhost:8000

### Метод 2: Node.js
```bash
cd /workspaces/diploma/src
npx http-server
```

### Метод 3: VS Code Live Server
1. Инсталирайте "Live Server" extension
2. Десен бутон върху `index.html`
3. "Open with Live Server"
