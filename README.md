# 💾 Управление на външни запаметяващи устройства

Дипломна работа за придобиване на професионална квалификация по специалност **„Икономическа информатика"** (код 4820101), професия „Икономист-информатик" (код 482010).

Проектът изследва видовете външни запаметяващи устройства (USB, SSD, HDD, SD, CD/DVD), рисковете при тяхната употреба и мерките за защита на данните. Включва теоретичен анализ, сравнителни таблици, казуси от реалния свят и уеб представяне с интерактивни графики.

> 🌐 **Онлайн версия:** [diploma.isaewwxx.dev](https://diploma.isaewwxx.dev)

---

## 🖥️ Уебсайт

Статичен сайт без фреймуърк, изграден с HTML, CSS и vanilla JavaScript. Всички графики и таблици се генерират динамично чрез Chart.js — без външни файлове с данни.

| Страница | Описание |
|---|---|
| 🏠 **Начало** | Въведение, хронология, сравнения на устройства, файлови системи, матрица на рисковете, казуси |
| 📊 **Графики** | 8 интерактивни диаграми + 17 таблици с данни от изследването |
| 👤 **Автор** | Информация за автора и проекта |
| ✉️ **Контакт** | Форма за обратна връзка (Formspree) |

**Използвани библиотеки:** [Chart.js](https://www.chartjs.org/) за визуализация · [Lucide](https://lucide.dev/) за икони

**Покрити стандарти:** ISO/IEC 27001, NIST SP 800-88, GDPR

## 🚀 Стартиране локално

Клонирайте репозиторието и пуснете статичен сървър от `web/`:

```bash
git clone https://github.com/isaewwxx/diploma.git
cd diploma/web
npx http-server
```

Или отворете `web/index.html` с [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) във VS Code.
