# Управление на външни запаметяващи устройства

Дипломна работа за придобиване на III степен на професионална квалификация по специалност „Икономическа информатика" (код 4820101), професия „Икономист-информатик" (код 482010).

Проектът изследва видовете външни запаметяващи устройства (USB, SSD, HDD, SD, CD/DVD), рисковете при тяхната употреба и мерките за защита на данните. Включва теоретичен анализ и уеб представяне с интерактивни графики.

## Уебсайт

Статичен сайт без фреймуърк, изграден с HTML, CSS и vanilla JavaScript.

| Страница | Описание |
|---|---|
| **Начало** | Въведение в темата, ключови факти и статистики |
| **Графики** | Интерактивни диаграми с данни от изследването (Chart.js) |
| **Автор** | Информация за автора |
| **Контакт** | Форма за обратна връзка |

Използвани библиотеки: [Chart.js](https://www.chartjs.org/) за визуализация, [Lucide](https://lucide.dev/) за икони.

## Стартиране

Клонирайте репозиторието и пуснете статичен сървър от `src/`:

```bash
git clone https://github.com/isaewwxx/diploma.git
cd diploma/src
npx http-server
```

Или отворете `src/index.html` с [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) във VS Code.

За директен достъп до онлайн версията (Cloudflare Pages):

https://diploma-a4l.pages.dev/
