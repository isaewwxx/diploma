## 🎯 За проекта

Този проект изследва управлението на външни запаметяващи устройства, включително техните **видове**, **рискове** и **методи за защита**. Комбинира теоретичен анализ с практическа разработка на информативен уебсайт.

- Видове външна памет
  - USB Flash drives

- Мерки за защита
  - Криптиране

  - Резервни копия
  - Политики за достъп
- Организационна практика и работа с флаш памет в корпоративна среда

## 🖥️ Стартиране

За стартиране на локален сървър използваме Node.js инструменти.

Вариант 1: Чрез `npx` (препоръчително)

```bash
cd src
npx http-server
```

Вариант 2: VS Code Live Server

1. Отворете папката в VS Code.
2. Инсталирайте разширението Live Server.
3. Кликнете десен бутон върху `index.html` -> Open with Live Server.
4. Отворете браузър на http://localhost:8080 (или порта, посочен от терминала).
