# 🖼️ Assets директория

Тази директория съдържа всички медийни файлове и ресурси за уебсайта.

## 📁 Предложена структура

```
assets/
├── images/
│   ├── devices/          # Изображения на устройства
│   │   ├── usb-flash.jpg
│   │   ├── ssd.jpg
│   │   ├── hdd.jpg
│   │   ├── sd-card.jpg
│   │   └── dvd.jpg
│   ├── icons/           # Икони
│   │   └── favicon.ico
│   └── bg/              # Фонове
│       └── hero-bg.jpg
├── fonts/               # Шрифтове (опционално)
└── docs/                # Допълнителни документи
```

## 📝 Препоръки за изображения

### Формати
- **JPEG** - За снимки и изображения с много цветове
- **PNG** - За изображения с прозрачност
- **SVG** - За икони и векторна графика
- **WebP** - Съвременен формат с добро сжатие

### Оптимизация
- Компресирайте изображенията преди качване
- Използвайте адекватна резолюция (не прекалено висока)
- Препоръчителни размери:
  - Hero изображения: 1920x1080px
  - Икони на устройства: 500x500px
  - Favicon: 32x32px, 64x64px

### Инструменти за оптимизация
- [TinyPNG](https://tinypng.com/) - Компресиране на PNG/JPEG
- [Squoosh](https://squoosh.app/) - Универсален оптимизатор
- [SVGOMG](https://jakearchibald.github.io/svgomg/) - SVG оптимизация

## 📜 Авторски права

- Всички изображения трябва да имат подходящ лиценз за използване
- Посочете източника на изображенията в документацията
- За комерсиална употреба проверете лиценза

## 💡 Съвети

1. **Именуване на файлове**
   - Използвайте малки букви
   - Заменете интервали с тирета: `usb-flash-drive.jpg`
   - Използвайте описателни имена

2. **Организация**
   - Групирайте по тип (устройства, икони, фонове)
   - Създайте под-директории при нужда

3. **Alt текст**
   - Винаги добавяйте описателен `alt` атрибут в HTML
   - Помага за SEO и достъпност

## 🔗 Използване в HTML

```html
<!-- Локално изображение -->
<img src="assets/images/devices/ssd.jpg" alt="SSD устройство">

<!-- С lazy loading -->
<img src="assets/images/devices/hdd.jpg" alt="HDD устройство" loading="lazy">

<!-- Responsive изображение -->
<img 
  srcset="assets/images/devices/usb-flash-small.jpg 400w,
          assets/images/devices/usb-flash-large.jpg 800w"
  sizes="(max-width: 600px) 400px, 800px"
  src="assets/images/devices/usb-flash.jpg" 
  alt="USB Flash устройство">
```
