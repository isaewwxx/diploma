# 💾 Уебсайт - Управление на външни запаметяващи устройства

Този директория съдържа всички файлове за уебсайта на дипломния проект.

## 📁 Структура

```
src/
├── index.html          # Начална страница
├── graphics.html       # Страница с графики и визуализации
├── contact.html        # Контактна страница с форма
├── author.html         # Страница за автора
├── css/
│   ├── style.css       # Основни стилове
│   └── responsive.css  # Responsive дизайн
├── js/
│   ├── main.js         # Основна функционалност
│   ├── charts.js       # Chart.js графики
│   └── contact.js      # Валидация на контактна форма
└── assets/
    └── README.md       # Информация за ресурси
```

## 🚀 Стартиране

### Вариант 1: Python HTTP сървър

```bash
cd src
python3 -m http.server 8000
```

После отворете браузър на: http://localhost:8000

### Вариант 2: Node.js http-server

```bash
cd src
npx http-server
```

### Вариант 3: VS Code Live Server

1. Инсталирайте разширението "Live Server" във VS Code
2. Кликнете десен бутон върху `index.html`
3. Изберете "Open with Live Server"
